Shopping Cart System. BSIT 1-1


This program is a console-based shopping cart system developed in C# using object-oriented programming concepts.

It allows users to:

•Browse the available products in the store

•Add selected items to the shopping cart

•Verify if items are still in stock

•Automatically update quantity for repeated selections

•Calculate the total cost and apply any discounts

•Display the remaining stock after completing the purchase



Additional features may be added in the future.


Feel free to try it out!
